# 맛집 추천 리스트

## 1. 마쯔무라돈까스 본점

- 위치 : 서울특별시 도봉구 노해로63길 84 지하 1층
- 가격대: 로스돈까스 - 10,500 / 히레돈까스 - 11,000
- 돈까스 짱 부드럽고 양도 많아요. 완전 추천!!  
  대신 4시 이후에 가면 원하는 돈까스가 품절일 수 있다는 사실.

## 2. 우차이

- 위치: 서울특별시 동대문구 회기동 경희대로4길 5 지하 1층
- 가격대: 짬뽕 - 8,000 / 짜장면 - 6,000
- 가성비도 좋은데 '짬뽕'이 매콤하니 아주 맛있다.  
  매우니까 꼭 밥 추가해서 먹도록!  
  짜장면은 짭짤해서 계속 먹게됨.  
  짜장면은 양이 조금 적어서 면 추가 하거나 밥 추가해서 먹어야 할듯.
